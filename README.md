# InstanTube
Superfast way to download Youtube videos!

It works for Linux, but code seamlessly adjusts to Windows environments, requiring only minimal modifications, such as editing the download paths.

# Dependencies

```bash
pip install pytube
pip install tk
pip install pillow
```
